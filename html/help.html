Notes


DDC project files generated by this version are no longer compatible with Viper's Toolbox 2.0.

This is because I implemented new filtertypes which aren't supported by the original version of Viper's Toolbox.

You can still generate *.vdcprj files that are compatible with this older version. To do that, please have a look in the 'Export' submenu and use the option called 'Project File v1.0'. Keep in mind that the filtertypes of all calibration points are set to 'peaking' by force, as this is the only type of filter supported in the original version.
