Notes


DDC project files using filtertypes other than 'Peaking' generated by this version are not compatible with Viper's Toolbox 2.0.

This is because these filtertypes are not supported by the original version of Viper's Toolbox.

Of course you can still generate *.vdcprj files that are compatible with this older version. To do that, just make sure that all filtertypes are set the 'Peaking' or use the option called 'Project File v1.0' in the 'Export' submenu. Keep in mind that if you use the export option, the filtertypes of all calibration points are set to 'Peaking' by force.
