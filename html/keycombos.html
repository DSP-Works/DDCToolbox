Key Shortcuts


Project:

Ctrl-O: Open project file

Ctrl-S: Save project file

Ctrl-E: Export as VDC


Edit:

Ctrl-N: Add point

Ctrl-R: Remove selected point(s)

Ctrl-Z: Undo

Ctrl-Shift-Z: Redo

Alt-Z: View undo history


View:

Ctrl-H: Hide graph

Ctrl-Print: Take screenshot of graph


Help:

F1: Shortcuts


Note: Double-click a cell to edit it.
